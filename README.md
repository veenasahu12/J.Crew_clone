# J.Crew_clone

### Project Description
J.Crew is an E-Commerce Website offering an assortment of Women's , Men's and Kid's apparel and Home accessories . It is an Individual Project created in 4 days .

## Tech Stack Used : -

- #### Languages
  - `HTML`
  - `CSS`
  - `JavaScript ES6`
  - `React`
  - `Redux`
  - `Chakra-UI`

#### CSS frameworks
  - `BootStrap`
